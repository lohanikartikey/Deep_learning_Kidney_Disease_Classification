# Deep_learning_Kidney_Disease_Classification

## Workflow
  1. Update config.yaml
  2. Update secrets.yaml [optional]
  3. Update params.yaml
  4. Update entity
  5. Update the config manager in src config
  6. Update the components
  7. Update the pipeline
  8. Update main.py
  9. Update dvc.yaml
  10. Update app.py
